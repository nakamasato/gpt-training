    Versions

    python 3.12.7
poetry 1.8.3

    Tools

    1. asdf

      asdf local python <version>
asdf local poetry <version>

    2. poetry

      poetry install

    Run

    Run a script

    poetry run python src/examples/agent_react.py

    Run an app

    poetry run streamlit run src/projects/apps/streamlit/app.py
